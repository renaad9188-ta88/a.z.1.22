# دليل إضافة صور البنر

## كيفية استبدال الصور

### الطريقة 1: استخدام صور من الإنترنت (Unsplash)
الصور الحالية تستخدم Unsplash. يمكنك تغيير URLs في ملف `lib/carousel-images.ts`

### الطريقة 2: استخدام صور محلية (موصى به)

1. أنشئ مجلد للصور:
   ```
   public/images/carousel/
   ```

2. أضف صورك:
   - `border.jpg` - صورة المعبر جابر / الحدود الأردنية السورية
   - `arrivals.jpg` - صورة القادمون
   - `departures.jpg` - صورة المغادرون
   - `travelers.jpg` - صورة المسافرين

3. عدّل ملف `lib/carousel-images.ts`:

```typescript
border: {
  url: '/images/carousel/border.jpg', // بدلاً من URL Unsplash
  title: 'المعبر جابر',
  subtitle: 'نقطة العبور الرئيسية',
  description: 'خدمات عبور سريعة وآمنة',
},
```

## مواصفات الصور الموصى بها:

- **الحجم**: 1200x600 بكسل (أو نسبة 2:1)
- **الصيغة**: JPG أو WebP
- **الحجم**: أقل من 500KB لكل صورة
- **الجودة**: عالية الوضوح

## نصائح للصور:

1. **صورة المعبر جابر**: صورة للمعبر أو الحدود الأردنية السورية
2. **صورة القادمون**: صورة لمسافرين قادمين أو استقبال
3. **صورة المغادرون**: صورة لمسافرين مغادرين أو سفر
4. **صورة المسافرين**: صورة لعائلات أو مسافرين

## مصادر صور مجانية:

- Unsplash: https://unsplash.com
- Pexels: https://pexels.com
- Pixabay: https://pixabay.com
